Proyecto Interactivo con Teachable Machine

Descripción del Proyecto

Este proyecto utiliza Teachable Machine y JavaScript para crear una aplicación web interactiva que permite al usuario realizar clasificaciones en tiempo real a través de la cámara de su computador. El objetivo es demostrar de manera práctica y visual cómo las tecnologías de Machine Learning pueden ser aplicadas en tareas del día a día, promoviendo la curiosidad y el aprendizaje en el campo de la inteligencia artificial.
